*** INSTALLATION ***
- Vérifiez que votre fournisseur d'accès supporte au moins le PHP 4.3 et MySQL 4.
- Copiez tous les fichers contenus dans l'archive sur votre ftp, dans le répertoire
 de votre choix (pour la suite, appelons ce répertoire 'forum').
- Changez les permissions sur le fichier 'forum/install.php', et mettez l'attribut
 CHMOD à 755 (permissions maximales pour la lecture, l'écriture et l'exécution).
- Rendez vous à l'adresse de votre forum ('forum'), l'installation débutera automatiquement.
- Suivez scrupuleusement les étapes spécifiées à l'écran.
- Une fois l'installation terminée, vous aurez accès au panneau d'administration pour
continuer la configuration de votre forum.

*** MISE A JOUR ***
- Copiez tous les fichiers contenus dans l'archive par dessus votre installation de
 Connectix Boards (vous devez écraser les fichiers qui sont en double).
- Rendez-vous sur la page install.php de votre forum, la mise à jour se lancera
 automatiquement.

ATTENTION !! :  Il est préférable de supprimer le fichier 'install.php' et le dossier
'install' après avoir installé ou mis à jour le forum.
